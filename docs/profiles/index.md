# Profiles

This section documents the available profiles for both systems and home configurations. Profiles provide reusable sets of configuration modules that can be shared across multiple systems or user accounts.

## [System Profiles](system/index.md)

System profiles are used to configure the base operating system, hardware support, and system-level services.

- **[Base](system/base.md)**: Common settings for all systems.
- **[Client](system/client.md)**: Configuration for client systems connecting to the NAS.
- **[Server](system/server.md)**: Media, networking, and sharing services.
- **[AI Server](system/ai-server.md)**: Dedicated AI and machine learning services.
- **[Workstation](system/workstation.md)**: Complete desktop environment and software.
- **[Limine UEFI](system/limine-uefi.md)**: Limine bootloader configuration.

## [Home Profiles](home/index.md)

Home profiles are used to configure the user environment, shell tools, and desktop preferences via Home Manager.

- **[Base](home/base.md)**: Core shell environment and tools.
- **[Desktop](home/desktop.md)**: Appearance and GUI application settings.
- **[Server](home/server.md)**: Optimized environment for server management.
- **[Workstation](home/workstation.md)**: Combined desktop and development tools.
