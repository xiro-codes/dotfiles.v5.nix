# MPD

Music Player Daemon (MPD) and ncmpcpp client configuration.

## Options

### `local.mpd.enable`

*   **Type:** `Boolean`
*   **Default:** `false`
*   **Description:** Enables MPD and ncmpcpp.

### `local.mpd.path`

*   **Type:** `String`
*   **Default:** `"/media/Music"`
*   **Description:** Path to the music directory for MPD to serve.
