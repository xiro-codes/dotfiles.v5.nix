# Hyprpaper

Hyprpaper wallpaper daemon configuration for Hyprland.

## Options

### `local.hyprpaper.enable`

*   **Type:** `Boolean`
*   **Default:** `false`
*   **Description:** Enables hyprpaper service.

### `local.hyprpaper.wallpapers`

*   **Type:** `List of Paths`
*   **Default:** `[ ]`
*   **Description:** A list of wallpaper paths to preload for Hyprpaper.
