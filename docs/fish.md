# Fish

Fish shell configuration with VI keybindings, zoxide, and eza abbreviations.

## Options

### `local.fish.enable`

*   **Type:** `Boolean`
*   **Default:** Determined by if the system shell is fish (`isDefaultShell`).
*   **Description:** Enables fish shell configuration if it is the system shell. This includes setting up `eza`, `zoxide`, `trash-cli`, `fastfetch`, shell abbreviations, and interactive shell initialization.
